# Air Quality Index (AQI) Prediction

## Project Overview
Developed an end-to-end AQI prediction system using large-scale European environmental data to analyze air quality patterns and predict AQI levels.

## Data Description
- 357,000+ European air quality records
- Environmental and geospatial features

## Tools & Technologies
- Python
- Machine Learning (Random Forest)
- Data Cleaning & Feature Engineering

- Cleaned and preprocessed large datasets
- Performed feature engineering and geospatial enrichment
- Trained Random Forest regression models

## Key Outcomes
