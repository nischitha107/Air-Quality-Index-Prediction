# Cyber Attack Detection in IoT Networks

## Overview
This project aims to detect cyber attacks in IoT networks using deep learning techniques. A real-time intrusion detection framework was designed to improve security monitoring in IoT environments.

## Business Problem
IoT networks are highly vulnerable to cyber threats due to large-scale connectivity and limited device security. Early detection of intrusions is critical to minimize operational and financial risks.

## Dataset
- KDD Intrusion Detection Dataset
- Network traffic features representing normal and malicious activity

## Tools & Technologies
- Python
- CNN-LSTM Deep Learning Models
- Mutual Information for Feature Selection
- Flask (Web Application Framework)
- Data Visualization

## Methodology
- Applied feature selection to reduce dimensionality and improve efficiency
- Designed CNN-LSTM models to capture spatial and temporal patterns
- Developed a real-time intrusion detection web application
- Visualized model predictions and performance metrics

## Key Results
- Improved detection efficiency while reducing computational complexity
- Successfully classified normal vs. malicious network traffic
- Built an interactive system for real-time monitoring

## Key Learnings
- Deep learning for cybersecurity applications
- Feature selection techniques for high-dimensional data
- Deploying analytical models using web frameworks
